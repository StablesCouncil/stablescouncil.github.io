Minima archive mirror

This Council endpoint publishes archive_*.raw.dat exports for operators who want to bootstrap or re-sync a Minima archive node. The node that produces these files is maintained with genesis coverage (see operator integrity checks in Stables governance docs).

Large downloads are served directly from Council infrastructure (not from Git). New dated files appear whenever operators run exportraw on the server (not automatic unless you add cron). Developer path: /devtools/minima-archive/.
