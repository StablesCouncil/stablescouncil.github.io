Immediate Work

Baseline the channels

Record the first weekly manual values for followers, members, posts, engagement, and views in Dashboard inputs.

Build the first content queue

Create five posts tied to Navi cards: launch explanation, Demo review, feedback intake, ambassador path, and Council/Charter draft.

Publish only what is true

Every post should name the correct Dapp phase and avoid implying Test or Prod mechanics before they exist.

Route feedback visibly

Questions and reactions should become Dashboard signals, feedback entries, StablesAgent gaps, or Navi card next steps.
