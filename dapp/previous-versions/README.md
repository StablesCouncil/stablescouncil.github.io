# Previous MiniDapp versions

This folder stores all previously published Stables MiniDapp packages.

## Retention policy

- Keep every historical `.mds.zip` version in this folder.
- Do not delete old versions; this is the permanent archive.
- Use versioned filenames so users can identify and recover specific builds.

## Current archive (examples)

| File | Meaning |
|------|--------|
| `Stables_v00.00.01.mds.zip` | First numbered showcase drop |
| `Stables_v00.00.02.mds.zip` | Second showcase drop (moved here when **`Stables_v00.00.00.00.03.mds.zip`** became **latest-version**) |
| `Stables_v0.01.01.mds.zip` | Older public baseline (legacy label) |

**Latest node install** is always **`../latest-version/Stables_v00.00.00.00.03.mds.zip`** until a newer row ships.
