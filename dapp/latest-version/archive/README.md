# Archived MiniDapp packages (demo)

Superseded demo **`.mds.zip`** files live here. **`../`** (parent folder) holds **only the current published** package for direct install.

- Do not delete historical zips; move them here when a new version ships.
- Filenames keep the published iteration label (e.g. `Stables_v0.0.0.3.44.mds.zip`).
- Release notes and dates: [`../VERSION_HISTORY.md`](../VERSION_HISTORY.md).
