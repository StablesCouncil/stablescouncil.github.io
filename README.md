# Stables

> **Money that is truly yours. Secure, Pseudonymous and Unstoppable.**

Stables is a global money platform designed for everyone. Built on the **[Minima](https://minima.global)** network, it provides universal access to stable value without traditional intermediaries, ensuring your money remains under your absolute control at all times.

## Site build (Eleventy, Council operators)

This folder is the working tree for **`https://stablescouncil.github.io/`**. Long-form pages that share site chrome are built with **[Eleventy](https://www.11ty.dev/)**.

1. **`npm install`** (once per clone).
2. Edit templates under **`src/`** (layouts and partials under **`src/_includes/`**; shared data in **`src/_data/`**).
3. Run **`npm run build`** before copying or pushing when anything under **`src/`** changed (templates that Eleventy compiles to root **`*.html`**).
4. Root static files (**`stables.css`**, **`assets/`**, **`CNAME`**, images) are edited directly in the repo; they are not generated by the default Eleventy config.
5. **The Playing Field** (**`playing_field.html`** at this folder’s root) is **hand-maintained**: edit that file for copy and structure; **`npm run build`** does not emit it. Shared chrome for future pages remains in **`src/_includes/`**.

Governance and UI contracts: **`0_handshake/handshake.md`** (GitHub Pages bullet), **`0_handshake/web_component_spec.md`** (GitHub Pages — Eleventy). Push workflow: **`handover_document.md`** and the nested clone under **`3_archive/…/stablescouncil.github.io/`** when that is your remote.

## The Vision

We believe that financial freedom should not be a privilege. Stables leverages the power of Minima to create a payment system that is:

*   **Secure**: Your assets are protected by the fundamental principles of a truly decentralized network.
*   **Pseudonymous**: Privacy is a right. Transact freely without compromising your personal identity.
*   **Unstoppable**: Designed to be resilient, borderless, and always accessible, no matter where you are in the world.
