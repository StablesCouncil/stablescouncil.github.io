# Stables – Central Brand Assets Index

This directory is the **single source of truth** for visual assets used across all Stables productions
(presentations, MiniDapp, docs, social, etc.).

Please copy the approved masters from `2_current/assets/brand_masters/` into this folder using the filenames below.

| Slot / Usage        | Expected filename    | Source master (for you)                  | Notes                                      |
|---------------------|----------------------|------------------------------------------|--------------------------------------------|
| Hero wordmark       | `logo-wordmark.png`  | `master_wordmark_transparent.png`       | Main logo in heroes / primary lockups.    |
| Symbol / icon       | `logo-symbol.png`    | `master_symbol.png`                      | Icon / app symbol / favicon source.       |
| Social banner       | `twitter-header.png` | `master_twitter_header.png`              | For social headers or wide hero usage.    |

All projects (e.g. `presentation/`, `dapp/`) should reference assets from `../brand/assets/…` instead of
keeping their own copies, unless we explicitly create derived/exported variants.
